# Solutions

## Overview
This repository contains solutions organised across four folders, each addressing a separate question.

---

## Question 1
This question focused on SDTM DS Domain creation.

### Files
- **01_create_ds_domain.R** — R code used to create the DS domain
- **output_create_ds_domain.log** — Output log confirming the script ran with no errors
- **sdtm_dataset.csv** — Final generated SDTM dataset

---

## Question 2
This question focused on ADaM ADSL dataset creation.

### Files
- **create_adsl.R** — R code used to create the ADSL dataset
- **adsl.csv** — Final generated ADSL dataset
- **output_create_adsl.log** — Output log confirming the script ran with no errors

---

## Question 3
This question focused on generating high-quality tables and data visualizations.

### Files
- **01_create_ae_summary_table.R** — R code for the AE summary table
- **02_create_visualizations.R** — R code for the data visualizations
- **output_create_ae_summary_table.log** — Output log confirming the table script ran with no errors
- **output_create_visualizations.log** — Output log confirming the visualization script ran with no errors
- **data_visualizations/** — Folder containing all generated tables and visualizations

---

## Question 4
This question utilized LangChain to develop a generative AI clinical trial data assistant.
The Jupyter Notebook contains both the code creating and running the agent, as well as the test script at the very bottom running 3 test queries. The outputs of this code are in the notebook directly but can also be run by installing all required packages and inserting a Claude Anthropic API Key.
