# Solutions

## Overview
This repository contains solutions organised across four folders, each addressing a separate question.

---

## Question 1
This question focused on tackling SDTM DS Domain creation. To evaluate the R code used, look through the file 01_create_ds_domain.R. It's corresponding output log denoting no errors is in file output_create_ds_domain.log. The final generated sdtm dataset can be found in sdtm_dataset.csv.

---

## Question 2
This question focused on tackling ADaM ADSL dataset creation. The file with the corresponding R code is in create_adsl.R, and the generated dataset is in adsl.csv. To confirm no errors occured during running, check the output log at output_create_adsl.log.

---

## Question 3
This question focused on the ability to generate high-quality tables and data visualizations. The code for the two parts are in 01_create_ae_summary_table.R and 02_create_visualizations.R. To see the tables and visualizations, please look inside the folder data_visualizations inside question_3_tlg. Lastly, the output logs showing no run errors are called output_create_ae_summary_table.log and output_create_visualizations.log. 

---

## Question 4
This question utilized LangChain to develop a genAI clinical trial data assistant. The file clinical_trial_assistant_agent.ipynb is a Jupyter Notebook that contains both the code creating and running the agent, as well as the test script at the very bottom running 3 test queries. The outputs of this code are in the notebook directly but can also be run by installing all required packages and inserting a Claude Anthropic API Key.
